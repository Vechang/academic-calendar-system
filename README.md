# Academic Calendar Management System

## Description
This is a web-based system used to manage and display academic events like exams, holidays, and presentations.  
Admin can add, update, and delete events, while users can view them.

---

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL (XAMPP)

---

## Prerequisites
- XAMPP (Apache & MySQL)
- Web Browser

---

## Installation Steps
1. Download or clone the project  
2. Move the folder to `htdocs` inside XAMPP  
3. Start Apache and MySQL  
4. Open phpMyAdmin and create a database named `aca_calendar`  
5. Import the `aca_calendar.sql` file  
6. Open browser and go to:  
   `http://localhost/project-folder-name`
